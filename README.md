# Galaxy Digital Geofeed

This feed is encoded in RFC 8805 format.

This data feed provides approximate geo locations for a subset of Galaxy Digital IP space and the data included in this feed may be used only for improving the accuracy of Geo:IP lookups.

## Zscaler Internet Access Disaster Recovery PAC File

This Proxy Auto-Configuration (PAC) file maintains a curated allowlist of critical business domains required for continuity of operations during Zscaler Internet Access service disruptions. The domains included in this list have been vetted and approved by the Galaxy Digital Security team and are specifically designated for disaster recovery scenarios where the Zscaler Global Safe list may be unavailable.
